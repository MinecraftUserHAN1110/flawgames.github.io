FlawYoutubeAPI is a plugin that connects Youtube and Minecraft. If you hit a certain letter on Youtube, you can make the set command work.

It should be noted that it works with only a few config settings.

Now we'll show you how to put this into action.

Config: 

api-key:
google-api-key:

live-chat-id:

command:
  one-command:
    # command from youtube
    from:
    # command send to console
    to:
  two-command:
    # command from youtube
    from:
    # command send to console
    to:
  three-command:
    # command from youtube
    from:
    # command send to console
    to:
  four-command:
    # command from youtube
    from:
    # command send to console
    to:
  five-command:
    # command from youtube
    from:
    # command send to console
    to:

Here's how the script works:

We'll tell you what to put in each.

api-key and google-api-key are the same letters, and require the key of the Google application.

live-chat-id means the key you need to share your screen. Simply put, live broadcasting key.

The command can be set to 5 types, and / is not required. You can put in a command that you type on YouTube for from, and a command that works on the server for to.
